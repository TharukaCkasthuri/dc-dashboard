# dc-dashboard
This is an interactive dashboard developed using d3.js , dc.js and crossfilter.js 

1. clone and download the project.
2. go to data folder and run htt_server.py file
3. change the chart.js script path in index.html file with your local script file path
4. go to template folder and run the index.html file.
5. see the dashboard on your favorit browser.
